# Polina's Cafe Billing System

Консольний додаток на мові програмування C# для автоматизації розрахунку та обліку замовлень у кафе "Polina's Cafe".

## Основний функціонал
* **Додавання страв (Add Item):** Введення назви страв (від 3 до 20 символів) та їхньої ціни (>0). Обмеження замовлення — до 5 страв.
* **Видалення страв (Remove Item):** Інтерактивний вибір страви за номером для її видалення зі зсувом паралельних масивів.
* **Додавання чайових (Add Tip):** Підтримка трьох режимів: відсоток від суми, фіксована сума або повне скасування чайових.
* **Виведення чека (Display Bill):** Автоматичний розрахунок проміжної суми (Net Total), ПДВ (GST 5%), чайових та підсумкової суми до сплати.
* **Очищення чека (Clear All):** Повне скидання поточного стану чека та чайових для нового клієнта.
* **Робота з файлами (Save/Load to CSV):** Можливість зберегти поточний чек у файл CSV та зчитувати його назад. Назва файлу обмежена довжиною від 1 до 10 символів, а самі файли зберігаються безпосередньо в кореневу папку проєкту.

## Технічні особливості
* Реалізовано без використання класів (виключно на базі масивів).
* Застосовано захисне програмування (Defensive Programming) — додаток повністю захищений від "вильотів" при некоректному введенні даних користувачем (текст замість цифр, завеликі значення тощо).
